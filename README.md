# Automobile-Sales-Dashboard
Power BI Dashboard Project
Automobile Sales Analytics Dashboard

Project Overview:
This project focuses on analyzing automobile sales data using Power BI. 
The dashboard provides insights into revenue performance, market distribution, customer preferences, and vehicle performance metrics.

Dataset Description
The dataset contains 10,000 records and includes the following important columns:

- Car Name
- Price
- Units Sold
- Fuel Type
- Transmission
- Mileage
- Horsepower
- Engine CC
- Resale Value
- Country Name
- Year

Data Cleaning Process
The following data cleaning steps were performed:

- Removed unnecessary columns to improve performance
- Replaced regional names with country names for better geographical analysis
- Handled missing values in numeric columns
- Ensured correct data types (numeric, categorical, date)
- Created calculated column for Profit estimation

Dashboard Pages

Page 1 – Executive Overview
- KPI Cards used to show Total Revenue, Units Sold, Average Price and Profit.
- Line Chart used to show Sales Growth Trend over time.
- Donut Chart used to show Fuel Type preference.
- Bar Chart used to show Top Selling Car Models.

Page 2 – Sales & Market Distribution
- Map Visual used to analyze global revenue distribution.
- Column Chart used to compare revenue by country.
- Scatter Plot used to analyze Transmission efficiency vs demand.

Page 3 – Vehicle Performance Insights
- Bar Chart used to compare Horsepower and Units Sold.
- Column Chart used to analyze Engine Capacity impact on revenue.
- Donut Chart used to understand Loan Adoption rate.
- Line Chart used to analyze Resale Value trend.

Business Insights
- High engine capacity vehicles generate higher revenue.
- Certain markets contribute more to overall sales.
- Fuel type preference affects demand patterns.
- Loan approval plays a role in customer purchasing decisions.
